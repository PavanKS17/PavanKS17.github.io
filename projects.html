Deep dives into the distributed systems, machine learning pipelines, and agentic workflows I have engineered for production.

01 // End-to-End MedTech Diagnostic Platform

The Problem: Manual clinical reviews for cervical imagery were bottlenecking patient diagnostics. Physical hardware capture devices lacked a streamlined connection to modern, high-accuracy inference models.

The Architecture: Bridged the physical-to-cloud gap by engineering a secure pipeline. Images are ingested via concurrent FastAPI microservices, processed by custom Vision Transformers (ViT), and returned to an intuitive clinician dashboard.

Vision Transformers (ViT) FastAPI Docker Edge-to-Cloud

The Impact

  • Reduced manual clinical review time by 40%.
  • Achieved state-of-the-art segmentation accuracy.
  • Solo-architected the entire product lifecycle from research to deployment.

Scale & Metrics

  • Processes 1M+ daily telemetry events.
  • Supports 10,000+ concurrent users.
  • Maintains 99.9% system uptime.

02 // Global Telemetry Backend

The Problem: A massive influx of IoT and device telemetry data required a highly available backend capable of ingesting, processing, and storing events without dropping packets or timing out under load.

The Architecture: Engineered a robust Java/Spring Boot backend hosted on Azure. Implemented aggressive SQL query optimization and connection pooling to handle high-volume transactional data efficiently against the MySQL database.

Java Spring Boot MySQL Azure

03 // Enterprise Code Collaboration Platform

The Architecture: Built automated, event-driven data pipelines utilizing AWS SQS for message queuing and AWS Lambda for serverless compute. This decoupled architecture allows for independent scaling of the ingestion and processing layers.

The Scale: Successfully ingests and processes over 5TB+ of monthly repository metadata, enabling seamless transactional workflows for 200+ distinct code repositories.
